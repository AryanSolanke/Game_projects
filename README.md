# Gambling games
A command-line based game built using python.

## Features
1. Menu-driven CLI interface
2. Jackpot number gambling game
3. Balance management with input validation

## Tech Stack
1. Python 3
2. Standard python libraries

## Disclamer
* This project is for practice only.
No real money is used.

## How to run 
1. Make sure python is installed.
2. Clone the repository:[Gambling games](https://github.com/AryanSolanke/Game_projects)
3. Run the game.
